# 💊 경구약제 객체 탐지 프로젝트

### 프로젝트 기간
2025.05.22 - 2025.06.10

---
### 프로젝트 개요
사진 속에 있는 최대 4개의 알약의 이름(클래스)과 위치(바운딩 박스)를 검출하는 모델을 구현하고, 성능을 지속적으로 개선해나가는 것

### 주요 기능
- YOLO 모델 기반 객체 탐지
- 탐지된 바운딩 박스 좌표 기반으로 객체 Crop 및 메타데이터 생성
- ResNet 모델을 활용한 알약 제품 분류
